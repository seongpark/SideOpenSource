# SideOpenSource
강릉명륜고등학교 통합 온라인 플랫폼 사이드

# 환경
* PHP 7
* Apache2
* MySQL

## Table 
* alert : 회원 알림 저장
* dday : 디데이 기능 저장
* lucky : 사이드 가입 이벤트 결과 저장
* member : 회원 정보 저장
* pwOTP : 비밀번호 찾기 1회용 OTP 저장
* student : 모바일 학생증 정보 저장
* umbrella : 우산대여제 내역 저장
* umbrellaOpen : 우산대여제 시작 여부 저장
* umbrellaList : 우산 상태 저장
* umbrellaBlack : 우산대여제 대여제한 내역 저장
